# Manifesto

No queremos construir software que haga más cosas. Queremos construir software que ayude mejor a las personas.

Flowly no existe para captar atención. Existe para devolver tiempo, claridad y capacidad operativa.

La IA es una herramienta de colaboración, no una autoridad. Las mejores empresas del futuro no serán las que tengan más IA, sino las que mejor colaboren con ella.
