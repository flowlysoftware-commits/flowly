# Non-Negotiable Principles

1. El dominio empresarial es el centro.
2. Los Business Objects son la fuente de verdad.
3. La IA nunca posee el dominio.
4. Todo debe ser observable.
5. Todo debe ser explicable.
6. Todo debe ser versionable.
7. Ningún componente es privilegiado.
8. El usuario conserva el control.
9. La arquitectura prevalece sobre la implementación.
10. Flowly se construye para durar décadas.
