# Flowly Constitution

La Constitución resume qué es Flowly, qué no debe ser y qué principios no puede romper.

## Principios centrales

1. El dominio empresarial es el centro.
2. La IA ayuda, pero nunca sustituye el juicio humano.
3. Toda acción importante debe ser observable, explicable y gobernada.
4. La arquitectura prevalece sobre la implementación.
5. Flowly existe para ayudar a las personas a trabajar mejor.
