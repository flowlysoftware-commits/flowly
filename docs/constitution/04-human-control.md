# Human Control

Flowly puede observar, recomendar, planificar y ejecutar, pero las decisiones relevantes pertenecen a las personas o a políticas explícitamente delegadas.

## Regla

Toda acción crítica requiere autorización, política o aprobación humana.

## Consecuencia

El Companion propone. El usuario decide. El Execution Engine ejecuta solo cuando la decisión está aprobada.
