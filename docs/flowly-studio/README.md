# Flowly Studio

Flowly Studio es el entorno interno para diseñar partes de Flowly OS antes de escribir código.

## Modos

- **Builder Mode:** diseño manual guiado de Business Objects, Capabilities, Workflows, Policies y Apps.
- **Architect Mode:** blueprints de arquitectura para módulos completos antes de convertirlos en diseños concretos.

## Flujo recomendado

1. Diseñar el artefacto.
2. Guardar en Supabase.
3. Revisar Generator Preview.
4. Exportar JSON si hace falta.
5. Convertir el diseño aprobado en código real.

## SQL

Ejecutar `supabase/flowly_studio.sql`.
