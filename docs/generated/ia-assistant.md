# IA Assistant

Módulo generado para el Companion de Flowly.

## Incluye

- Perfil del Companion.
- Avatar.
- Objetivos.
- Misiones diarias y semanales.
- Recompensas.
- Logros.
- Niveles.
- Experiencia.
- Estado de ánimo.
- Actividad.

## Ruta

`/ia-assistant`

## Estado

Primera versión instalable y revisable generada desde Flowly Studio V2.
