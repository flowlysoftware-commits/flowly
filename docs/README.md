# Flowly Docs

Flowly Docs es el sistema de conocimiento vivo de Flowly OS.

Reúne arquitectura, ingeniería, implementación, dominio, capacidades, APIs, SDK, Marketplace, Companion, seguridad, DevOps y operación en un único lugar versionado junto al código.

La documentación debe evolucionar junto al código. Ningún cambio relevante está terminado si deja Flowly Docs desactualizado.
