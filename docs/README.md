# Flowly Docs

Flowly Docs es el centro de conocimiento vivo de Flowly OS.

Este espacio no está pensado como una carpeta de documentos sueltos, sino como una base preparada para convertirse en un módulo nativo de Flowly con buscador, IA, relaciones entre conceptos, enlaces al código y exportación futura a PDF, Word o Markdown.

## Estructura principal

- **Constitution**: esencia, principios y manifiesto de Flowly.
- **Architecture Bible**: visión completa del Living Business Operating System.
- **Reference Architecture**: plano técnico del Kernel, Engines, Capabilities y dominio.
- **Engineering Handbook**: reglas para construir Flowly.
- **Implementation Blueprint**: siguiente fase, orientada a implementación real.
- **Domain Catalog**: Business Objects oficiales.
- **Capability Catalog**: capacidades reutilizables.
- **API / SDK / Marketplace**: ecosistema externo.
- **Companion**: especificación del Companion OS.

## Regla

La documentación debe evolucionar junto con el código. Si cambia el sistema, cambia Flowly Docs.
