# Agent Operating Protocol

Este libro define cómo deben trabajar Codex, Copilot u otros agentes de IA dentro de Flowly OS.

El objetivo es evitar que los agentes improvisen, dupliquen motores o creen parches aislados. Cada agente debe leer la documentación viva de Flowly, entender la arquitectura existente y ejecutar cambios de forma verificable.
